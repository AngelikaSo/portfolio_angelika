YoYoga Studio

Landing page of the yoga studio

Problem, Ideation, Wireframing, Figma Design, HTML5, CSS3, JavaScript, Netlify, Git, GitHub

View Live

View Code

The Main Goal

Build a landing page for a fictional yoga (YoYoga) studio in order to initially present my coding and web design skillset.

Execution

  1. I have focused on building a responsive webpage, by using media queries CSS3 module, in order to make this landing page accesible for different screen-size devices.
  2. I have planned and designed the landing page in Figma, then transfered the design to HTML5, styled in CSS3, and finally added JavaScript functions to make it more dynamic.
  3. In this project I have used Git for version control, and GitHub to manage Git repositories. Finally I have hosted the project on Netlify to run the webpage.
