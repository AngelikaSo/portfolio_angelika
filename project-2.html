The Main Goal

The goal of my personal website was to present both my abilities as well as my personality.

Execution

  1. Planned and designed the personal portfolio in Figma, then decided to use HTML, CSS, and JavaScript to execute the project and make it run live. Built a responsive webpage, by using media queries CSS3 module, to be accessible from different screen-size devices.
  2. Created a simple, user friendly interface, which shows my personality, and preference for minimalism in design, in order to highlight prepared projects.
  3. Decided to use black and white as the main colors with a light blue accent, in order to emphasize the content of the portfolio. Bold sections' titles and simple linear illustrations are designed to attract the attention of the visitor.
  4. Focused on designing the landing page with simple navigation to make sure the visitor can find every information without problem.
