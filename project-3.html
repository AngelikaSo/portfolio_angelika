The Main Goal

Inspired by my interest in planing, and developing a daily routine I decided to automize my paper planer, and create a simple to use web application for everyday use.

Execution

  1. I wanted the visual interface to be simple, well designed, yet attractive for user. I decided to use purple as a main color which is vibrant and stylish.
  2. Additionally user can choose between dark and light mode to display the application.
  3. User can add their task, check-mark each time the task is completed, edit or delete the task (CRUD functionality). Tasks are saved in local storage.

User Testing

As soon the app was running I asked few friends to test the functionallity out. I received their feedback that I took into account and improved the app functionallity.

    Features I was able to implement so far:
  • Responsive app design - app can be displayed on different screen-size devices.
  • Bigger font size for tasks.
  • Better color of the crossed-out tasks.
  • Better UI for the longest tasks.
  • Breaking the long words in order to display task in the designated area.
    Features to improve:
  • Breaking point of the longest words.
  • Visible scrollbar (nice UI and UX).
